    Class ContribucionCongreso

    Representa una contribución científica presentada en un congreso. Extiende la funcionalidad de GeneralClass para incluir detalles específicos del evento. *

    Example

    const ponencia = new ContribucionCongreso(
    "IA en la Educación", ["Juan Pérez"], ["IA", "EdTech"],
    "Resumen del estudio...", "2024", 20, "IEEE",
    "Congreso Mundial de Tecnología", "Madrid"
    );
    console.log(ponencia.obtenerIEEE());
