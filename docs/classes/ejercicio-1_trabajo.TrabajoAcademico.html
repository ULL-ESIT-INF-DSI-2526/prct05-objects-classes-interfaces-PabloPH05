    Class TrabajoAcademico

    Representa un trabajo de índole académica oficial, como tesis o proyectos de fin de carrera. Hereda de GeneralClass y añade metadatos sobre el grado académico y el departamento. *

    Example

    const tesis = new TrabajoAcademico(
    "Optimización de Redes Neuronales",
    ["Laura García"],
    ["Deep Learning", "Eficiencia"],
    "Estudio profundo sobre...",
    "2024",
    150,
    "Universidad Politécnica",
    "Tesis Doctoral",
    "Departamento de Inteligencia Artificial"
    );
    console.log(tesis.obtenerIEEE());
